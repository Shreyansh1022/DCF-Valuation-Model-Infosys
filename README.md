# DCF-Valuation-Model-Infosys
This project focuses on valuing Infosys using the Discounted Cash Flow (DCF) method. The model estimates the intrinsic value of the company based on its projected future cash flows.
Project Structure
The model is built in Microsoft Excel and includes:
Assumptions Sheet (growth rates, margins, discount rate)
Historical Financial Data
5-Year Financial Forecast
DCF Model (Free Cash Flow, Discounting, Terminal Value)
Methodology
The valuation process includes:
Forecasting revenue and operating performance
Calculating Free Cash Flow (FCF)
Discounting future cash flows using WACC
Estimating Terminal Value using the Gordon Growth Model
Deriving Enterprise Value
Key Features
5-year financial forecasting
Dynamic model linked with assumptions
Calculation of Free Cash Flow (FCF)
Terminal Value estimation
Enterprise valuation output
